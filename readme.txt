■概要

  nokaを起動するだけのプラグインです。

  noka
    https://github.com/betonetojp/noka

  起動時に最新のReleasesをチェックし、更新されたzipが添付されていれば
  それをダウンロードして上書きします。
  常に最新のnokaを使用することができます。
